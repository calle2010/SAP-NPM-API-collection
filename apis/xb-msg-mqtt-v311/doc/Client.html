pause()

The client shall pause receiving any data, as the broker is sending faster than the application can consume.

Before using this method consider changing one single channels flow only.

As soon as possible call the client to resume.

The whole connection is paused. Hence, it affects for example also confirms as well as connection or channel exceptions.
